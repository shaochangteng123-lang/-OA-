## 📋 授权配置步骤

### 步骤1：访问飞书开放平台

1. 打开浏览器，访问：**https://open.feishu.cn/app**
2. 使用**管理员账号**登录（需要有应用管理权限）

### 步骤2：找到您的应用

1. 在应用列表中，找到 **"工作日志"** 应用
2. 点击进入应用详情页

### 步骤3：配置可用范围（关键步骤）

根据飞书官方文档，可用范围配置在 **"用户管理"** 中：

#### 方式A：通过"用户管理"配置（推荐）

1. **进入用户管理页面**
   - 左侧菜单 → **"用户管理"** → **"可用范围"**
   - 或：左侧菜单 → **"用户管理"** → **"可见范围"**

2. **选择可见范围类型**

   **选项1：全部成员可见（最简单，推荐）**
   ```
   - 选择 "全部成员可见"
   - 点击 "保存"
   - ✅ 所有飞书用户都可以看到并使用此应用
   ```

   **选项2：指定成员可见**
   ```
   - 选择 "指定成员可见"
   - 点击 "添加成员" 或 "选择成员"
   - 搜索并添加：邵长腾
   - 可以继续添加其他需要访问的用户
   - 点击 "保存"
   ```

   **选项3：指定部门可见**
   ```
   - 选择 "指定部门可见"
   - 点击 "添加部门" 或 "选择部门"
   - 选择需要访问的部门（如：行政部、财务部、项目部等）
   - 点击 "保存"
   ```

#### 方式B：通过"版本管理与发布"配置

1. 左侧菜单 → **"应用发布"** → **"版本管理与发布"**
2. 创建新版本或编辑现有版本
3. 在版本配置中找到 **"可用范围"** 或 **"可见范围"** 设置
4. 配置可见范围（同上）
5. 保存并发布版本

#### 方式C：通过"基础信息"配置

1. 左侧菜单 → **"基础信息"** → **"凭证与基础信息"**
2. 查找 **"可用范围"** 或 **"可见范围"** 设置
3. 配置可见范围（同上）

### 步骤4：发布应用（重要）

配置可见范围后，**必须发布应用**才能生效：

1. 左侧菜单 → **"应用发布"** → **"版本管理与发布"**
2. 如果已有未发布的版本：
   - 点击 **"发布"** 或 **"提交审核"**
3. 如果没有版本：
   - 点击 **"创建版本"**
   - 填写版本号（如：1.0.0）
   - 填写版本说明
   - 点击 **"提交审核"** 或 **"发布"**

### 步骤5：等待配置生效

1. **等待时间**：配置保存并发布后，等待 **2-3 分钟** 让配置生效
2. **清除缓存**：建议清除浏览器缓存或使用无痕模式测试

### 步骤6：验证授权

1. **在飞书工作台查看**
   - 使用"邵长腾"账号登录飞书
   - 打开飞书工作台
   - 查看是否能看到"工作日志"应用图标

2. **测试访问**
   - 点击"工作日志"应用
   - 确认可以正常打开，不再提示权限错误

## 🔍 如果找不到配置入口

### 方法1：查看权限的API文档

1. 在权限管理页面，找到 **"管理应用可见范围"** 权限行
2. 点击右侧的 **"相关API/事件"** 按钮
3. 查看API文档，了解如何通过API配置

### 方法2：联系飞书技术支持

- 在飞书开放平台页面查找 **"帮助"** 或 **"客服"** 入口
- 咨询：如何配置应用可见范围/可用范围

### 方法3：检查应用状态

确保应用状态为 **"已发布"**：
- 如果应用未发布，需要先发布应用
- 应用发布后，可见范围配置才会生效

## 🆘 常见问题

### Q1: 找不到"可用范围"配置入口？

**解决方案：**
1. 尝试在 **"版本管理与发布"** 中配置
2. 尝试在 **"基础信息"** 中查找
3. 查看 **"管理应用可见范围"** 权限的 **"相关API/事件"**
4. 联系飞书技术支持

### Q2: 配置后仍然提示"没有使用权限"？

**解决方案：**
1. ✅ 确认配置已保存
2. ✅ 确认已发布应用版本（**非常重要**）
3. ✅ 等待 2-3 分钟让配置生效
4. ✅ 清除浏览器缓存后重试
5. ✅ 使用无痕模式测试
6. ✅ 确认使用的是正确的飞书账号（邵长腾）

### Q3: 如何确认配置是否生效？

**检查方法：**
1. 在飞书工作台查看是否能看到应用
2. 尝试点击应用，看是否能正常打开
3. 查看服务器日志，看是否有权限相关错误

### Q4: 应用状态显示"未发布"怎么办？

**解决方案：**
1. 进入 **"版本管理与发布"**
2. 创建新版本或编辑现有版本
3. 填写版本信息
4. 点击 **"提交审核"** 或 **"发布"**
5. 等待审核通过（如果需要）
