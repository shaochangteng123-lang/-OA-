# 发票批量上传和拖拽上传功能说明

## 功能概述

发票上传组件现在支持批量上传和拖拽上传，提升用户体验和操作效率。

## 新增功能

### 1. 批量上传
- **功能**：一次可以选择多个PDF文件进行上传
- **使用方法**：
  1. 点击上传区域
  2. 在文件选择对话框中，按住 Ctrl（Windows）或 Command（Mac）键选择多个文件
  3. 或者在文件选择对话框中，选择第一个文件后，按住 Shift 键选择最后一个文件，批量选中
  4. 点击"打开"按钮

### 2. 拖拽上传
- **功能**：可以直接将文件拖拽到上传区域
- **使用方法**：
  1. 从文件管理器中选择一个或多个PDF文件
  2. 拖拽到上传区域（虚线框区域）
  3. 松开鼠标，文件自动开始上传和识别

### 3. 视觉反馈
- **拖拽悬停**：当文件拖拽到上传区域上方时，区域会高亮显示
- **上传图标**：使用更大的上传图标（48px），更容易识别
- **提示文字**：
  - 主提示："拖拽文件到此处或点击上传"
  - 副提示："支持批量上传，一次可选择多个PDF文件"

### 4. 识别进度提示
- **开始识别**：显示 "正在识别发票 [文件名]..."
- **识别成功**：显示 "[文件名] 识别成功！"
- **识别失败**：显示 "[文件名] 识别失败：[错误原因]"

## 技术实现

### 组件配置
```vue
<el-upload
  drag          <!-- 启用拖拽上传 -->
  multiple      <!-- 启用多选 -->
  accept=".pdf"
  :limit="maxFiles"
>
```

### 关键属性
- `drag`：启用拖拽上传功能
- `multiple`：启用多文件选择
- `accept=".pdf"`：限制只能选择PDF文件
- `:limit="maxFiles"`：限制最大上传数量（默认5个）

### 样式优化
- 拖拽区域高度：180px
- 虚线边框：2px dashed #d9d9d9
- 悬停效果：边框颜色变为主题色
- 拖拽悬停：背景色变为浅色主题色

## 使用限制

1. **文件格式**：仅支持PDF文件
2. **文件大小**：单个文件不超过5MB
3. **文件数量**：
   - 基础报销：最多5个文件
   - 大额报销：最多10个文件
   - 商务报销：最多10个文件

## 注意事项

1. 批量上传时，每个文件会独立进行OCR识别
2. 识别过程是异步的，可能需要几秒钟
3. 如果某个文件识别失败，不会影响其他文件
4. 建议一次上传不超过5个文件，以获得更好的体验

## 浏览器兼容性

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

拖拽上传功能依赖于 HTML5 Drag and Drop API，在现代浏览器中均有良好支持。
