# 配置飞书应用可见范围 - 详细步骤

## 📋 前提条件

✅ **已完成**：权限"管理应用可见范围"（admin:app.visibility）已开通

## 🎯 配置步骤

### 步骤1：找到可见范围配置入口

根据您的截图，权限管理页面已经打开。现在需要找到可见范围的配置入口：

#### 方式A：通过左侧菜单
1. 在左侧菜单中，查找以下位置：
   - **"基础信息"** → 可能包含可见范围设置
   - **"应用能力"** → 查看是否有可见范围相关选项
   - **"应用发布"** → "版本管理与发布" → 在发布版本时配置

#### 方式B：在当前权限管理页面
1. 找到"管理应用可见范围"权限行
2. 点击右侧的 **"相关API/事件"** 按钮
3. 查看API文档，了解如何通过API配置

### 步骤2：配置可见范围

找到配置入口后，选择以下方式之一：

#### 选项1：全部成员可见（最简单，推荐）
- 选择 **"全部成员可见"**
- 点击 **"保存"**
- ✅ 所有飞书用户都可以看到并使用此应用

#### 选项2：指定成员可见
- 选择 **"指定成员可见"**
- 点击 **"添加成员"** 或 **"选择成员"**
- 搜索并添加以下用户：
  - **邵长腾**
  - 其他需要访问的用户
- 点击 **"保存"**

#### 选项3：指定部门可见
- 选择 **"指定部门可见"**
- 点击 **"添加部门"** 或 **"选择部门"**
- 选择需要访问的部门（如：行政部、财务部、项目部等）
- 点击 **"保存"**

### 步骤3：发布应用版本（如果需要）

如果配置后需要发布新版本：
1. 点击左侧菜单 **"应用发布"** → **"版本管理与发布"**
2. 创建新版本或更新现有版本
3. 填写版本说明
4. 点击 **"提交审核"** 或 **"发布"**

### 步骤4：验证配置

1. **等待配置生效**：保存后等待 1-2 分钟
2. **测试登录**：
   - 使用"邵长腾"账号登录飞书
   - 在飞书工作台中查看是否能看到"工作日志"应用
   - 尝试点击应用，确认可以正常访问
3. **检查权限**：
   - 如果仍然提示"没有使用权限"，请检查：
     - 配置是否已保存
     - 是否已发布新版本（如果需要）
     - 等待时间是否足够（2-3分钟）

## 🔍 如果找不到配置入口

如果按照上述步骤找不到可见范围配置入口，可以尝试：

1. **查看飞书文档**：
   - 访问：https://open.feishu.cn/document/
   - 搜索"应用可见范围"或"admin:app.visibility"
   - 查看API文档和配置说明

2. **联系飞书支持**：
   - 在飞书开放平台页面查找"帮助"或"客服"入口
   - 咨询如何配置应用可见范围

3. **通过API配置**：
   - 查看"管理应用可见范围"权限的"相关API/事件"
   - 使用API调用配置可见范围

## 🆘 常见问题

### Q1: 找不到"可见范围"配置入口？
**解决方案：**
- 尝试在"版本管理与发布"中配置
- 查看"相关API/事件"了解API调用方式
- 联系飞书技术支持

### Q2: 配置后仍然提示"没有使用权限"？
**解决方案：**
1. 确认配置已保存
2. 确认已发布新版本（如果需要）
3. 等待 2-3 分钟让配置生效
4. 清除浏览器缓存后重试
5. 确认使用的是正确的飞书账号登录

### Q3: 如何添加更多用户？
**解决方案：**
- 在可见范围配置中添加更多成员
- 或选择"全部成员可见"
