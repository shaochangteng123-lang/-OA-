## 🚀 快速开始

### 1. 环境要求

- Node.js >= 18
- npm >= 9
- 飞书企业账号（用于 OAuth 认证）

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量

复制 `.env.example` 到 `.env` 并配置：

```bash
# 飞书应用配置
FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
FEISHU_REDIRECT_URI=http://localhost:8899/api/auth/callback

# 数据库配置
DATABASE_PATH=./data/worklog.db

# 会话配置
SESSION_SECRET=your_session_secret

# 前端地址
FRONTEND_URL=http://localhost:8899

# 服务器端口
PORT=3000
```

### 4. 启动开发服务器

```bash
npm run dev
```

- 前端：http://localhost:8899
- 后端 API：http://localhost:3000

### 5. 构建生产版本

```bash
npm run build
npm run start
```

### 6. Docker 部署

```bash
# 快速启动
./docker-start.sh

# 或使用 docker compose
docker compose up -d --build
```

## 📖 使用指南

### 用户角色与权限

| 角色 | 权限说明 |
|------|---------|
| **super_admin** | 超级管理员，拥有所有权限 |
| **admin** | 管理员，可管理用户和系统配置 |
| **user** | 普通用户，可使用所有业务功能 |
| **guest** | 访客，只读权限 |

**配置管理员：** 编辑 `server/config/admins.json`

```json
{
  "superAdmins": [
    { "name": "张三", "comment": "系统管理员" }
  ],
  "admins": [
    { "name": "李四", "comment": "部门管理员" }
  ]
}
```

### 常用功能

#### 📝 编写工作日志

1. 登录系统后，默认进入"今日日志"页面
2. 使用 Notion 风格编辑器编写日志
3. 支持 Markdown 语法和斜杠命令（`/`）
4. 自动保存草稿，防止数据丢失
5. 可关联项目，方便后续查询

#### 📅 管理日历事件

1. 进入"日历"页面
2. 支持日/周/月三种视图切换
3. 点击空白区域创建新事件
4. 支持循环事件（使用 rrule）
5. 可从事件库快速添加常用事件

#### 💰 提交报销申请

1. 进入对应的报销类型页面
2. 填写报销信息
3. 上传发票图片（支持 OCR 自动识别）
4. 提交审批
5. 跟踪审批进度

#### 📁 管理项目

1. 进入"项目管理"页面
2. 查看项目列表和状态
3. 点击项目查看详情
4. 更新项目进度
5. 完成后进行项目封存

## 🔧 开发指南

### 目录结构

```
yulilog-worklog/
├── src/                    # 前端源码
│   ├── views/              # 页面组件
│   ├── components/         # 可复用组件
│   ├── stores/             # Pinia 状态管理
│   ├── router/             # 路由配置
│   ├── types/              # TypeScript 类型
│   └── utils/              # 工具函数
├── server/                 # 后端源码
│   ├── routes/             # API 路由
│   ├── middleware/         # 中间件
│   ├── services/           # 外部服务集成
│   ├── db/                 # 数据库
│   └── config/             # 配置文件
├── user/                   # 用户文档（本目录）
└── data/                   # 数据文件（SQLite）
```

### 技术栈

**前端：**
- Vue 3 + TypeScript
- Pinia（状态管理）
- Vue Router（路由）
- Element Plus（UI 组件）
- TipTap（富文本编辑器）
- TanStack Query（数据请求）

**后端：**
- Express + TypeScript
- better-sqlite3（数据库）
- express-session（会话管理）
- 飞书 OAuth（认证）

### 代码规范

- 使用 TypeScript 严格模式
- 使用 Vue 3 Composition API
- 遵循 ESLint 规则
- 代码注释使用中文
- 组件名使用 PascalCase
- 文件名使用 kebab-case


## 🐛 常见问题

### 1. 登录失败

**问题：** 点击登录后无法跳转或提示错误

**解决方案：**
- 检查飞书应用配置是否正确
- 确认 `FEISHU_REDIRECT_URI` 与飞书后台配置一致
- 查看浏览器控制台和服务器日志

### 2. 数据库错误

**问题：** 提示数据库连接失败或表不存在

**解决方案：**
- 检查 `DATABASE_PATH` 配置
- 确保 `data/` 目录存在且有写权限
- 删除 `data/worklog.db` 重新初始化

### 3. 会话过期

**问题：** 频繁需要重新登录

**解决方案：**
- 检查 `SESSION_SECRET` 是否配置
- 确认浏览器允许第三方 Cookie
- 检查服务器时间是否正确

### 4. Docker 部署问题

**问题：** Docker 容器启动失败

**解决方案：**
- 检查端口 8899 是否被占用
- 查看容器日志：`npm run docker:logs`
- 确保 `.env` 文件存在且配置正确
