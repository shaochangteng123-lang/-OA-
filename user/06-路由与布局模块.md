# 06 - 路由与布局模块

## 模块概述

路由与布局模块负责应用的页面路由管理、导航守卫、页面布局框架和权限控制。

## 主要功能

### 1. 路由管理
- 页面路由配置
- 懒加载优化
- 路由元信息管理
- 动态路由生成

### 2. 导航守卫
- 登录状态检查
- 权限验证
- 页面访问控制
- 重定向逻辑

### 3. 布局系统
- 主布局框架
- 侧边栏导航
- 顶部导航栏
- 面包屑导航

### 4. 页面切换
- 路由过渡动画
- 页面加载状态
- 错误页面处理

### 路由元信息
```typescript
interface RouteMeta {
  title?: string              // 页面标题
  requiresAuth?: boolean      // 是否需要登录
  permission?: string         // 所需权限
  roles?: string[]            // 允许的角色
  icon?: string               // 菜单图标
  hidden?: boolean            // 是否在菜单中隐藏
  breadcrumb?: boolean        // 是否显示面包屑
}
```

## 导航守卫

### 全局前置守卫
```typescript
router.beforeEach(async (to, from, next) => {
  const authStore = useAuthStore()

  // 设置页面标题
  document.title = to.meta.title
    ? `${to.meta.title} - YuliLog`
    : 'YuliLog 工作日志系统'

  // 检查是否需要登录
  if (to.meta.requiresAuth && !authStore.isAuthenticated) {
    return next({ name: 'Login', query: { redirect: to.fullPath } })
  }

  // 检查权限
  if (to.meta.permission && !authStore.hasPermission(to.meta.permission)) {
    ElMessage.error('您没有访问该页面的权限')
    return next({ name: 'Home' })
  }

  next()
})
```

### 全局后置守卫
```typescript
router.afterEach((to, from) => {
  // 页面切换后的操作
  // 例如：关闭加载状态、记录访问日志等
})
```

## 使用示例

### 编程式导航
```typescript
import { useRouter } from 'vue-router'

const router = useRouter()

// 跳转到指定页面
router.push({ name: 'Calendar' })

// 带参数跳转
router.push({
  name: 'ProjectDetail',
  params: { id: '123' }
})

// 带查询参数跳转
router.push({
  path: '/projects',
  query: { status: 'active' }
})

// 返回上一页
router.back()
```

### 路由守卫使用
```typescript
// 在组件内使用路由守卫
import { onBeforeRouteLeave } from 'vue-router'

onBeforeRouteLeave((to, from) => {
  if (hasUnsavedChanges.value) {
    const answer = window.confirm('有未保存的更改，确定要离开吗?')
    if (!answer) return false
  }
})
```

### 动态面包屑
```typescript
// 根据路由生成面包屑
const breadcrumbs = computed(() => {
  return route.matched
    .filter(r => r.meta.title && r.meta.breadcrumb !== false)
    .map(r => ({
      title: r.meta.title,
      path: r.path
    }))
})
```

## 特色功能

### 1. 权限路由
- 根据用户权限动态生成路由
- 无权限路由自动隐藏
- 访问无权限页面自动拦截

### 2. 路由缓存
- 使用 `keep-alive` 缓存页面状态
- 支持自定义缓存策略
- 优化页面切换性能

### 3. 路由过渡
- 页面切换动画效果
- 加载状态提示
- 错误页面友好提示

### 4. 面包屑导航
- 自动生成面包屑
- 支持点击返回上级
- 可配置显示/隐藏

## 注意事项

1. **路由命名**：
   - 所有路由必须设置唯一的 `name`
   - 便于编程式导航和路由跳转

2. **权限控制**：
   - `meta.permission` 与后端权限保持一致
   - 前端权限检查仅用于 UI 展示
   - 后端必须有对应的权限验证

3. **懒加载**：
   - 大型页面组件必须使用懒加载
   - 避免首屏加载过大的 bundle

4. **路由嵌套**：
   - 保持路由层级简洁
   - 避免过深的嵌套层级
