# 14-系统设置模块

## 模块概述

系统设置模块提供系统级配置和用户个人偏好设置功能。包括系统管理入口、用户偏好配置、日历视图设置、颜色标签管理等功能。

## 主要功能

### 1. 系统设置页面
- 系统管理功能入口导航
- 用户管理入口
- 部门管理入口
- 事件库管理入口
- 预设方案管理入口
- 板块设置入口
- 系统信息展示

### 2. 用户设置页面
- 个人信息展示
- 日历偏好设置
- 颜色标签管理
- 账号操作（退出登录）

## 数据模型

### 用户偏好设置 (UserPreferences)

```typescript
interface UserPreferences {
  userId: string                    // 用户ID
  calendarViewMode: 'month' | 'week' | 'day'  // 日历默认视图
  weekDisplayDays: 5 | 7            // 周视图显示天数
  colorLabels: ColorLabel[]         // 颜色标签配置
  createdAt?: string               // 创建时间
  updatedAt?: string               // 更新时间
}
```

### 颜色标签 (ColorLabel)

```typescript
interface ColorLabel {
  id: string                        // 标签ID
  name: string                      // 标签名称
  color: string                     // 标签颜色（十六进制）
  order: number                     // 排序顺序
}
```

## 页面功能详解

### 系统设置页面

**路由**: 暂无独立路由（通过侧边栏访问）

**页面布局**:
- 页面头部：系统设置标题
- 设置卡片网格：响应式网格布局

**功能卡片**:

1. **用户管理卡片**
   - 图标：用户图标（蓝色）
   - 标题：用户管理
   - 描述：管理系统用户、角色分配和权限控制
   - 点击跳转：`/users`

2. **部门管理卡片**
   - 图标：办公楼图标（绿色）
   - 标题：部门管理
   - 描述：管理政府部门信息、简称和网站链接
   - 点击跳转：`/departments`

3. **事件库管理卡片**
   - 图标：文件图标（橙色）
   - 标题：事件库管理
   - 描述：管理全局事件库、事件类型和层级结构
   - 点击跳转：`/events`

4. **预设方案卡片**
   - 图标：集合图标（蓝色）
   - 标题：预设方案
   - 描述：管理项目预设模板和工作流配置
   - 点击跳转：`/presets`

5. **板块设置卡片**
   - 图标：网格图标（红色）
   - 标题：板块设置
   - 描述：管理预设板块分类和区块配置
   - 点击跳转：`/blocks`

6. **系统信息卡片**
   - 图标：信息图标（灰色）
   - 标题：系统信息
   - 内容：版本号、运行环境
   - 不可点击

**UI 特点**:
- 卡片悬停效果：向上平移 2px
- 响应式布局：移动端单列，桌面端多列
- 图标颜色区分：不同功能使用不同颜色
- 箭头指示：可点击卡片显示右箭头

### 用户设置页面

**路由**: 暂无独立路由（通过用户菜单访问）

**页面布局**:
- 页面头部：用户设置标题
- 设置区域：多个设置卡片

**功能区域**:

#### 1. 个人信息卡片
- 用户头像（80px）
- 用户名称
- 用户邮箱
- 角色标签（不同角色不同颜色）

**角色标签颜色**:
- `super_admin`: danger（红色）
- `admin`: warning（橙色）
- `user`: primary（蓝色）
- `guest`: info（灰色）

#### 2. 日历偏好设置
- **默认视图**：
  - 月视图
  - 周视图
  - 日视图
  - 使用单选按钮组

- **周视图天数**：
  - 5天（工作日）
  - 7天（全周）
  - 使用单选按钮组

- 修改后自动保存

#### 3. 颜色标签设置
- 标签列表展示
- 每个标签包含：
  - 颜色选择器
  - 标签名称输入框
  - 删除按钮
- 添加标签按钮（最多 10 个）
- 恢复默认按钮
- 至少保留 1 个标签

**标签操作**:
- 修改颜色：使用颜色选择器
- 修改名称：输入框失焦时保存
- 删除标签：点击删除按钮（至少保留 1 个）
- 添加标签：点击添加按钮（最多 10 个）
- 恢复默认：重置为系统默认标签

#### 4. 账号操作
- 退出登录按钮
- 点击后清除会话并跳转到登录页

## 数据库表结构（待实现）

### user_preferences 表

```sql
CREATE TABLE user_preferences (
  user_id TEXT PRIMARY KEY,
  calendar_view_mode TEXT NOT NULL DEFAULT 'week',
  week_display_days INTEGER NOT NULL DEFAULT 7,
  color_labels TEXT NOT NULL,  -- JSON 格式存储
  created_at TEXT NOT NULL,
  updated_at TEXT NOT NULL,
  FOREIGN KEY (user_id) REFERENCES users(id)
);
```

**字段说明**:
- `user_id`: 用户ID，主键，外键关联 users 表
- `calendar_view_mode`: 日历默认视图（month/week/day）
- `week_display_days`: 周视图显示天数（5 或 7）
- `color_labels`: 颜色标签配置，JSON 格式存储
- `created_at`: 创建时间
- `updated_at`: 更新时间

## 权限控制

### 系统设置页面
- **所有登录用户**: 可以访问系统设置页面
- **权限检查**: 各功能卡片跳转后由目标页面进行权限检查

### 用户设置页面
- **所有登录用户**: 可以访问和修改自己的偏好设置
- **数据隔离**: 用户只能查看和修改自己的设置

## 业务场景

### 典型使用流程

#### 系统管理员配置系统
1. 访问系统设置页面
2. 点击相应功能卡片
3. 进入具体管理页面
4. 完成配置后返回

#### 用户配置个人偏好
1. 点击用户头像或设置按钮
2. 进入用户设置页面
3. 修改日历视图偏好
4. 自定义颜色标签
5. 系统自动保存设置

## UI 设计特点

### 系统设置页面
- **卡片式导航**: 使用卡片展示各功能入口
- **图标区分**: 不同功能使用不同颜色图标
- **悬停效果**: 卡片悬停时向上平移
- **响应式布局**: 自适应不同屏幕尺寸

### 用户设置页面
- **分区展示**: 使用卡片分隔不同设置区域
- **即时保存**: 修改后自动保存，无需点击保存按钮
- **视觉反馈**: 保存成功后显示提示消息
- **颜色预览**: 颜色选择器实时预览效果

## 注意事项

1. **自动保存**: 用户设置修改后自动保存，无需手动点击保存按钮
2. **数据验证**: 颜色标签数量限制在 1-10 个之间
3. **默认值**: 新用户首次访问时使用系统默认配置
4. **权限检查**: 系统设置页面的各功能入口需要在目标页面进行权限检查
5. **响应式设计**: 页面需要适配移动端和桌面端
