# 飞书应用可见范围配置指南

## 一、在飞书开放平台配置应用可见范围

### 步骤1：访问飞书开放平台
1. 打开浏览器，访问：**https://open.feishu.cn/app**
2. 使用管理员账号登录

### 步骤2：找到您的应用
1. 在应用列表中，找到"工作日志"应用（或您的应用名称）
2. 点击进入应用详情页

### 步骤3：配置可见范围
1. 在左侧菜单中，点击 **"权限管理"** → **"可见范围"**
2. 选择可见范围类型：

#### 选项A：全部成员可见（推荐，最简单）
- 选择 **"全部成员可见"**
- 点击 **"保存"**
- ✅ 所有飞书用户都可以看到并使用此应用

#### 选项B：指定成员可见
- 选择 **"指定成员可见"**
- 点击 **"添加成员"**
- 搜索并添加以下用户：
  - **邵长腾**
  - 其他需要访问的用户
- 点击 **"保存"**

#### 选项C：指定部门可见
- 选择 **"指定部门可见"**
- 点击 **"添加部门"**
- 选择需要访问的部门（如：行政部、财务部、项目部等）
- 点击 **"保存"**

### 步骤4：验证配置
1. 配置保存后，等待1-2分钟让配置生效
2. 使用"邵长腾"账号登录飞书
3. 在飞书工作台中查看是否能看到"工作日志"应用
4. 尝试点击应用，确认可以正常访问

## 二、系统内部权限配置（已完成）

✅ 已将"邵长腾"添加到系统超级管理员配置文件：
- 配置文件位置：`server/config/admins.json`
- 配置内容：
```json
{
  "superAdmins": [
    {
      "name": "用户512390",
      "comment": "主要超级管理员"
    },
    {
      "name": "邵长腾",
      "comment": "超级管理员"
    }
  ]
}
```

## 三、常见问题

### Q1: 配置后仍然提示"没有使用权限"
**解决方案：**
1. 确认在飞书开放平台已正确保存配置
2. 等待2-3分钟让配置生效
3. 清除浏览器缓存后重试
4. 确认使用的是正确的飞书账号登录

### Q2: 如何添加更多用户？
**解决方案：**
- 在飞书开放平台的"可见范围"中添加用户
- 或在系统配置文件中添加用户（`server/config/admins.json`）

### Q3: 如何让所有用户都能访问？
**解决方案：**
- 在飞书开放平台选择"全部成员可见"
